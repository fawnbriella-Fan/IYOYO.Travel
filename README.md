# China Easy Travel

A Next.js travel website framework for foreign visitors traveling to China.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Main files

- `app/page.jsx` - homepage
- `app/cities/[city]/page.jsx` - dynamic city page
- `data/cities.js` - city content data
- `components/` - reusable components
